# Lawn Tweaks:
An edit of Quality of the Lawn that adds more bugfixes and addons for debugging, modding, and playing

Recommendation: Search /*LawnTweaks for comments on the changes in this decomp

- Better Debug Menu
- Better Healthbars
- More HUD (Toggleable) - More Plant/Zombie stats shown and seed packet timers
- Bugfixes
- Cool window titles that show your progression in a level
- Ice level with arrow key scrollable plant selection (trust its cool)
